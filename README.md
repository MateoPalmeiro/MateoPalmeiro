A lo largo de la carrera he trabajado en áreas bastante distintas. Las bases fueron algoritmos y estructuras de datos, arquitectura de computadores, sistemas operativos y redes. ## Conocimientos técnicos

### Ciberseguridad y criptografía
- Criptografía simétrica/asimétrica, funciones hash, firma digital, PKI, PGP/GPG, S/MIME, esteganografía, tarjetas inteligentes.
- Análisis de malware, análisis forense básico (fases, herramientas HW/SW).
- Auditorías de seguridad, OWASP Top 10, port scanning, fingerprinting, (D)DoS, sniffing.
- ISO/IEC 27000 y planificación básica de SGSI.

### Sistemas y redes
- Administración Linux/UNIX en producción: usuarios, grupos, LVM, RAID, ZFS, cifrado, PAM, LDAP, cron, servicios, logs.
- Virtualización con **Proxmox**: VMs, contenedores LXC, redes virtuales, almacenamiento.
- **Homeserver** propio: autohosting de servicios, DNS, backup, hardening.
- Redes: VLANs, trunking, EtherChannel, STP/RSTP, OSPF, BGP, IPv4/IPv6, subredes, monitorización.

### Desarrollo y automatización
- Scripting con Bash y Python para automatización de sistemas y análisis de datos.
- APIs REST, JSON/XML, Django, Docker, OAuth.
- SQL avanzado, administración de bases de datos (Oracle).
- Servicios distribuidos: sockets Java, RPC/Thrift, JDBC.
